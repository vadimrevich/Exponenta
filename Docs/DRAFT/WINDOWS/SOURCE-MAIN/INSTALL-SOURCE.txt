ИНСТАЛЛЯЦИЯ АДМИН-ПАКА «ЭКСПОНЕНТА»
=====================================

Подготовка инсталляционного файла осуществляется при помощи пакета InnoSetup. Актуальную его версию вы можете скачать с помощью менеджера пакетов Chocolatey.

В каталоге содержатся основные модули админ-пака «Экспонента» и файл описания InnoSetup Exponenta.iss. С помощью этого файла

- создаётся инсталлятор и деинсталлятор админ-пака;
- в проект добавляются все файлы с расширением *.exe, *.bat, *.cmd, *.reg
- создаются дополнительные скрытые каталоги и переменные окружения;
- добавляется скрипт восстановления minifilters при удалении Админ-пака;
- добавляются другие опции установки.

ПОРЯДОК СОЗДАНИЯ ИНСТАЛЛЯЦИОННОГО ФАЙЛА
=======================================

- Откройте файл Exponenta.iss двойным щелчком на нём в файловом менеджере;
- Замените в нём строку 
	Output=no 
на 
	Output=yes
- Замените  в строке
	OutputDir=C:\CODE\Compile
значение C:\CODE\Compile на свой допустимый целевой путь;
- Нажмите сочетание клавиш Ctrl+F9;
- Дождитесь окончания компиляции файлов;

Готовый файл поместите на публичный сервер для автоматической установки Админ-пака из Интернета.
