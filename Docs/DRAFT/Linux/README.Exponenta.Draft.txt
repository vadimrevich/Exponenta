Exponenta for Linux Admin Pack and Stealer
==================================

ОПИСАНИЕ
-----------------

Админ пак Exponenta for Linux предназначен для первоначальной настройки под управлением операционной системы Linux после её установки, с целью придания ей новых возможностей и уменьшения её защищённости по-умолчанию. Настройка включает в себя установка демонов и открытие удалённого доступа к компьютеру.

Пакет Exponenta for Linux не является клоном или гомологом пакета Exponenta for Windows, скорее, это его аналог со схожими функциями. Данный пакет написан «с нуля» той же группой программистов.

Стилер Exponenta for Linux предназначен для скрытой установки админ-пака Exponenta на компьютеры под управлением операционной системы Linux и запуска служб, которые первоначально или по соображениям безопасности постоянно отключены в разных версиях дистрибутивов Linux. (Примечание. Функционал стилера является экспериментальным и автором не проверялся на валидность). Файлы стилера распространяются отдельно.

ИНСТАЛЛЯЦИЯ И УДАЛЕНИЕ
------------------------------------------

Смотри соответствующий файл INSTALL.MD. Для запуска процесса установки или удаления требуются права суперпользователя Linux.

Для установки стилера эти права можно получить обманным путём, методом социальной инженерии, или воспользовавшись пакетом Metasploit.

---------------------------

Затем начал писаться модуль стилера, предназначенный только для удалённой установки админ-пака на «подотчётные компьютеры» администратора организации. Автор предупреждает о недопустимости криминального использования данных программ и распространения данных программ в скомпилированном виде, включая примеры распространения данных программ методами социальной инженерии. 

DESCRIPTION
---------------------

The administrator''s package of Exponenta is intended for initial control running the Windows/Linux/FreeBSD/MacOS X operating system (so far only wish list) after its installation, for the purpose of giving of new functions and reduction of its security to it by default. Control includes installation of services and opening of remote access to the computer.

Exponenta package for remote computer hacking and management. "Malware" is universal therefore the author does not specify his class. A nearest class in this category of malwares — Stealer. The software is designated by antiviruses as malware-gen. "Malware" is intended for the hidden installation the Admin Pack Exponenta on computers under control of the specified operating systems and start of services which originally or constantly for safety reasons are disconnected in different versions of these distribution kits. (Note. The functionality of a stealer is experimental and was not checked by the author for validity). Files of a stealer extend separately.
