AdminT Exponenta for Linux Plugin
===========================

ОПИСАНИЕ
-----------------

AdminT является полнофункциональным пакетом програмv и скриптов, расширяющих базовый функционал пакета Exponenta for Linux. Данный пакет может быть использован независимо от базового модуля Exponenta.

Данный универсальный админ пак предназначен для следующих целей:

- установки и предоставления облегчённого доступа к файлам по TFTP протоколу
- установка программы TigerVNC (TightVNC) и предоставление удалённого доступа к компьютеру по VNC протоколу

ИНСТАЛЛЯЦИЯ И УДАЛЕНИЕ АДМИН ПАКА
---------------------------------------------------------------

Смотри соответствующий файл INSTALL.MD

РАБОТА С АДМИН ПАКОМ
-------------------------------------

Особенностью работы данных программ является то, что они должны быть запущены на компьютере, к которому уже получен удалённый доступ. Это связано с тем, что часть программ из него являются интерактивными, и требуют для своей работы взаимодействия с пользователем. Программы запускаются из-под учётной записи root на компьютере. При удалённом доступе можно зайти под учётной записью root непосредственно или при помощи команды sudo.

Для запуска программ и скриптов из пакета необходимо:

1. Запустить терминальное окно Linux.
2. Зайти в систему под учётной записью «root».
3. Перейти в рабочий каталог пользователя «root» командой «cd ~».
4. Запустить модуль инициализации пакета Exponenta «exponenta_environment.sh».
5. Запустить в командной строке скрипты, просто набрав их в командной строке, нажав Enter и следуя инструкции их применения.
